# 选择器

> css 层叠样式表


```css
// 元素选择器
a{}

// 伪元素选择器(伪元素：一个真实存在的元素 )
::before{}

// 类选择器
.link{}

// 属性选择器
[type=radio]{}

// 伪类选择器(伪类：一个元素的状态)
:hover{}

// ID选择器
#id{}

// 组合选择器
[type=checkbox] + label{}

// 否定选择器
:not(.link){}

// 通用选择器
*{}

```


> 选择器的权重

首先看选择器的级别，级别一样看权重值

```
id选择器 #id{} +100

类 属性 伪类  +10

元素 伪元素  +1

其他选择器 +0

!important 优先级最高

元素属性优先级高(直接 style="")

权重相同 后写生效 
```
